# 📊 Noisy Time Series Dashboard

An interactive **Streamlit dashboard** for generating, visualizing, and analyzing **noisy time series data** with trend, seasonality, and randomness.

---

## 🚀 Features

- 📈 Generate synthetic time series data with:
  - Trend
  - Seasonality
  - Noise
- 📊 Interactive Plotly visualizations
- 📉 Rolling mean analysis
- 🔍 Time series decomposition
- 🚨 Anomaly detection (IQR method)
- 🔮 Moving average prediction
- 📂 Upload your own CSV data
- 📥 Download generated dataset
- 📊 Statistical summary (mean, std, etc.)
- 📦 Distribution plots (Histogram & Boxplot)

---

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- Statsmodels
## 📈 Dashboard Sections
Time Series Plot (with components)
Summary Statistics
Histogram & Box Plot
Rolling Mean Analysis
Scatter Plot
Time Series Decomposition
Dataset Preview
